## General Requirements

- Java 17 (tested with Amazon Corretto 17.0.10)
- Gradle 8.14.2 



## Small Description about how the application runs via the CLI

BUILD -> gradle clean build
RUN -> java -jar ./build/libs/schoolapp-0.0.1-SNAPSHOT.jar
